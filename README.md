Daily Deen — Islamic Spiritual Companion App
Built for the Quran Foundation Hackathon 2026 Daily Deen is a cross-platform mobile application designed to deepen your daily connection with Islam through reflection, knowledge, and community.

✨ Features
📖 Daily Wisdom Hub — Personalized Quranic verses and spiritual prompts, streak tracking, and daily task timeline
✍️ Reflection Journal — Seal your daily thoughts with intent tags and privacy controls (private or share to community)
🌿 Sanctuary of the Soul — Your spiritual growth map and journaling space on the Journey screen
📚 Seek Wisdom Library — Search thousands of scholarly resources; browse by Knowledge Domain (Family, Ethics, Finance)
🤖 Scholar AI Beta — Powered Spiritual Consultation chat with contextual Islamic rulings and Quranic references
🌍 Global Sanctuary — Community feed to read and appreciate anonymous reflections from fellow travelers
⚙️ Settings — Full account management, notification preferences, and app customization
🔌 API Integration
Integrated with the Quran Foundation APIs using OAuth2 (Client Credentials):

Secure token authentication via prelive-oauth2.quran.foundation
Content API for daily verses and scholarly resources
User API for streak tracking and reflection posting
🛠️ Tech Stack
Layer	Technology
Frontend	React + Vite
Styling	Vanilla CSS (Modern Sanctuary Design System)
Auth	OAuth2 Client Credentials (Quran Foundation)
Icons	Lucide React
Mobile-Ready	Capacitor-compatible structure
